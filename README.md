## üîß Usage Instructions

1. **Open CTk-Color-Picker**: After installation, launch the application.
   
2. **Select a Color**: Use the color wheel to find the color you want. Click on the color for it to appear in the selection box.

3. **Adjust Brightness**: Use the brightness slider to make your chosen color lighter or darker.

4. **Copy HEX Code**: Once you find the perfect color, click the copy button to get the HEX code in your clipboard. You can paste it anywhere you need.

5. **Save Your Settings**: CTk-Color-Picker allows you to save your selected colors for future reference.
